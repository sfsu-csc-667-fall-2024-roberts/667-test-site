0. Setup

Compilation

Run the following command from the root of your project to compile the Webserver (you may adjust the path separator for your operating system): 

javac server/WebServer.java

Record the document root

Have a folder/finder window open to the document root of this site so that we can see the filesystem changes as they occur.

Run the server

Run the following command from the root of your project to start the Webserver (you may adjust the path separator for your operating system): 

java server.WebServer PORT_NUMBER DOCUMENT_ROOT

1. Bad Request

Request

In your terminal, from the document root, run the following command to send a bad request to the server (you must have node installed): 

node app.js

2. GET Request

  1. GET existing page (200)
  2. GET missing page (404)

3. HEAD Request

  1. HEAD existing page (200)
  2. HEAD missing page (404)

4. PUT Request

  1. PUT a page (201)
  2. Check that page was created

5. DELETE Request

  1. DELETE a page (204)
  2. Check that page was deleted (404)

6. Simple Authentication

Open each page in an incognito tab to perform the following tests.

  1. GET secret page without credentials (401)
  2. GET secret page with correct credentials (200)
    Page content should appear here
  3. GET secret page with wrong credentials (403)
